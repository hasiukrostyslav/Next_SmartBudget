## Tech Stack

**Client:** React, Next, TailwindCSS.

**Server:** Next, Prisma.

**DataBase:** PostgreSQL.

## Project setup

```bash
# clone the project
$ git clone https://github.com/hasiukrostyslav/Next_SmartBudget.git

# go to the project directory
$ cd Next_SmartBudget

# install dependencies
$ npm install
```

## Build and run the project

```bash
# build
$ npm run build

# production mode
$ npm run start

# development mode
$ npm run dev
```
